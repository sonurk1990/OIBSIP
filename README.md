# Oasis Infobyte Data Science Internship Projects

This repository contains 5 end-to-end Data Science and Machine Learning projects completed as part of Oasis Infobyte Internship.

Each project includes:
• Data Cleaning
• Exploratory Data Analysis (EDA)
• Data Visualization
• Model Building
• Model Evaluation
• Prediction System

---

## Projects List

### Task 1 — Iris Flower Classification
Predict flower species using ML algorithms.

Models Used:
Logistic Regression
Decision Tree
Random Forest
SVM
KNN

---

### Task 2 — Unemployment Analysis with Python
Analyze unemployment rate trends using data visualization.

Key Analysis:
State wise unemployment
Covid impact analysis
Trend visualization

---

### Task 3 — Car Price Prediction using Machine Learning
Predict car price based on features like:
fuel type
engine size
horsepower
mileage

Models Used:
Linear Regression
Random Forest

---

### Task 4 — Email Spam Detection
Classify emails as spam or not spam using NLP.

Techniques:
Text preprocessing
TF-IDF
Naive Bayes / Logistic Regression

---

### Task 5 — Sales Prediction using Python
Predict product sales based on advertising budget.

Features:
TV marketing spend
Radio marketing spend
Newspaper marketing spend

Model Used:
Linear Regression
